# 💰 Expense Tracker (Python)

A simple and efficient **Expense Tracker** built using Python to help users manage their daily expenses, track spending habits, and maintain financial discipline.

---

## 🚀 Features

* ➕ Add new expenses
* 📋 View all expenses
* 🔍 Search/filter expenses
* 📊 Track total spending
* 💾 Data storage (file-based / in-memory depending on your implementation)

---

## 🛠️ Tech Stack

* **Language:** Python
* **Concepts Used:** File Handling, Functions, Loops, Conditionals

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/expense-tracker.git
```

2. Navigate to the folder:

```
cd expense-tracker
```

3. Run the program:

```
python expense_tracker.py
```
